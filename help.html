Arbitrary Precision Calculator – Help

Write your formulas as you're used to do in JavaScript, one per line, e.g. 4*(b1111^b1010)+xff*o77, i.e. bitwise XOR the binary values 1111 and 1010, then multiply the result with 4 (decimal), finally add the product of hexadecimal value ff and the octal value 77. You can also assign values to variables like in a = 2**128. Choose numeral system of results by choosing one of the radio buttons.
